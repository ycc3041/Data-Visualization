# Data-Visualization
This repository contains the implementation of several data visualization skills using different Python libraries.


## Prerequisites
* Python 3.6
* matplotlib 2.2.2
* plotly 1.12.9
* seaborn 0.9.0
* bokeh 0.13.0
* networkx 1.11
